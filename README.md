It updates package list, checks packages to be upgrade and displays notification on user's desktop.

Dependencies:
-------------
apt
coreutils
cron
grep
iputils-ping
ktsuss | sparky-su | menu
mount
sed
sparky-aptus-upgrade
sparky-remsu (>= 0.2.10)
sudo
yad

Install:
-------------
su (or sudo) 
./install.sh

Uninstall:
-------------
su (or sudo)
./install.sh uninstall
